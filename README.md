# Guardião da Floresta

Guardião da Floresta é um jogo de plataforma 2D desenvolvido na Unity como projeto acadêmico da disciplina de Game Development.

## Descrição

O jogo apresenta um personagem que deve atravessar fases de dificuldade progressiva, coletar diamantes, evitar obstáculos e chegar ao portal final.

## Mecânicas implementadas

- Movimento lateral
- Corrida com Shift
- Pulo
- Escalada
- Coletáveis
- Pontuação
- Sistema de vidas
- Obstáculos com dano
- Reinício da fase ao cair no vazio
- Portais para troca de fase
- Menu inicial
- Tela final
- Efeitos sonoros
- Música de fundo

## Controles

- A / D ou setas direcionais: movimentar
- Shift: correr
- Espaço: pular
- W / S ou setas para cima/baixo: escalar

## Fases

O jogo possui três fases com dificuldade crescente:

- Fase 1: introdução às mecânicas básicas
- Fase 2: aumento de obstáculos e plataformas
- Fase 3: maior desafio com mais pulos, escalada e perigos

## Tecnologias utilizadas

- Unity
- C#
- Assets 2D
- TextMeshPro
- Sistema de UI da Unity

## Como executar

Abra o projeto na Unity e execute a cena `MenuInicial`.

Também é possível executar a versão exportada do jogo pela pasta Build.
